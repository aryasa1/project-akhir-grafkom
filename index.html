Project Akhir Grafika Komputer

Deskripsi

Our final project is making 3D object of mini playground.
The object is swing, see saw, sand box with a bucket in there, and slider.
We use some scratched metalic texture for playground tools, and plastic texture for the bucket.
The plain using grass texture and we using box to make sky effect.
The see saw can move automactically.
